# Portfolio Documentation

Welcome to the portfolio documentation! This section contains detailed information about projects, technical implementations, and system architectures.

## Features

- **MDX Support**: Rich markdown with React components
- **Mermaid Diagrams**: Interactive flowcharts and diagrams
- **Code Highlighting**: Syntax highlighting for multiple languages
- **Search**: Full-text search across all documentation
- **Responsive Design**: Works on all devices

Explore the different sections:

## Technology Stack

This documentation is built with:

- **Nextra**: Next.js-based documentation framework
- **MDX**: Markdown with JSX support
- **Mermaid**: Diagram and flowchart generation
- **Next.js 14**: React framework with App Router
